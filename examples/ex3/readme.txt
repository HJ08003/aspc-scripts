This is an example on pscl and coda

Description of pscl: The absentee ballot outcomes contrasted with machine ballots, cast in Pennsylvania State Senate elections, selected districts, 1982-1993.

https://cran.r-project.org/web/packages/pscl/pscl.pdf



Description of coda: Provides functions for summarizing and plotting the
output from Markov Chain Monte Carlo (MCMC) simulations, as
well as diagnostic tests of convergence to the equilibrium
distribution of the Markov chain.

https://cran.r-project.org/web/packages/coda/coda.pdf
